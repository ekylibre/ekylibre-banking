# Ekylibre Banking

Allow to grab bank data for bank statement and payments from Nordigen

## Description

Work with https://ob.nordigen.com/

## QuickStart

Add into your .env NORDIGEN_SECRET_ID & NORDIGEN_SECRET_KEY

https://nordigen.com/en/account_information_documenation/integration/quickstart_guide/
